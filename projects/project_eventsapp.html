Description

This group assignment involved creating both a back-end API and a front-end for an app where users could post event details with an accompanying image, to a common page.

The main aim of the project was to learn about authentication and authorization, and to that end, the back-end makes use of credentials stored in a MongoDB database, with access control implemented via secret token passing.

My main contributions were the form and front-end functionality for adding/updating an event, and the authentication/authorization endpoints, including adding support for cookie-based tokens and encrypted passwords.

I also managed the two GitHub repositories on this project.
