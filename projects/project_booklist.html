Description

This group project involved using Node and Express to develop an API for accessing and modifying a book-list.

In order to test the API, we primarily made use of Insomnia, which allowed us to craft HTTP requests for each endpoint.

My contributions included the endpoints for search, count, ranged retrieval (for pagination), and marking a book as read/unread.

As a personal side-project, I also created a front-end which made use of the above API.
This also assisted the project as a whole by providing another way to test the endpoints and an easier way to enter test-data.
