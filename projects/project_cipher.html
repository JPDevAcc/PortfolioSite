Description

A sandbox for playing around with simple ciphers.

This project started off as an attempt to implement the classic Caesar Cipher where a message is encoded by shifting each character's alphabetical position by a secret key value (historically a shift of three, though there seems to be some debate as to whether it was to the left or right).

As an enhancement to this basic scheme, I added the ability to have a rolling key where the shift value is incremented by a second key value after each character. I also added the ability to chain ciphers together.

The HTML <template> tag was especially useful in this project, used to define templates for the active-cipher settings boxes.

* Updated version includes permutation (aka transposition) ciphers, bugfixes, UI improvements, unit tests, and more complete code commenting.
