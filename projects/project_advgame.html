~ The Project ~

Description

An adventure game in the style of classic command-driven interactive-fiction. This project makes heavy use of OOP classes to represent game entities such as rooms, items, and characters, and like the JavaScript Quiz Project uses a flexible file format to define the map and other game data.

Implemented features include container unlocking / opening, nestable containers, player inventory, consumables, and a rudimentary battle system.

Due to the one-week time constraint for this project, a lot of desired features were left out, such as an improved command parser, a more flexible map-definition format, support for images illustrating the player's current location, and a graphical compass and map to aid player navigation. Due to the modular nature of the code however, it would probably be relatively straightfoward to add these enhancements.
