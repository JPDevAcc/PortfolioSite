~ The Project ~

Description

This was the first project I carried out as part of the Developer Academy "Web Developer Bootcamp" and showcases my interest in retro-computing topics.
It was also an opportunity to practice my front-end skills and gain familiarity with the Bootstrap CSS framework.

I decided to use the Bootstrap 'grid' system, which is based on Flexbox, to create a three-column layout for all the pages. This system is based on dividing a container element into 12 equally wide column-widths, and then creating content-columns spanning these widths. In addition, Bootstrap classes were used to set when these columns would stack vertically on narrower viewports such as on mobile devices.

I also made the decision to use a dark theme for the site. This was largely a personal aesthetic choice but fitted the 'retro' theme quite well, although it did cause issues with excessive contrast when including images with white backgrounds.
