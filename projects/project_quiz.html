~ The Project ~

Description

This was a project to create a browser-based personality or knowledge quiz using JavaScript for game logic and dynamic content.

I decided early on that I would use some kind of data-file for the game data so that questions and other settings could easily be changed without needing to make modifications to the code. For this purpose, I decided to use an exported object from a JavaScript module file.

I also decided that I would support multiple quizzes on the same page, and settled on a Bootstrap carousel component for making the selection. In addition, I added support for both personality and knowledge quiz types, and also the ability to set non-uniform weights for scoring each answer.

Other planned features included a 'new game' button visible after completing a quiz, and a panel showing which questions had been answered and allowing the player to jump to any question, but due to time constraints, these were not implemented.
