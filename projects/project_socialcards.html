~ The Project ~

Description

A site where users can post cards to a common page with an image and accompanying message. Other users can then comment and add likes / dislikes for these posts. All state including the user-list is held locally on a per-browser basis, via localstorage.

This was the first project undertaken on the developer bootcamp which uses React instead of plain JavaScript, and allowed for a more rapid development process thanks to reactive updates and features like JSX.

My main contributions were the user registration page, complete with form validation, and the administration page where an 'admin' user could list/remove other users. I also provided a starting framework for the project with basic state handling.
